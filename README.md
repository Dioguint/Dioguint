## 📂 Projetos

### 🏥 [Sistema Hospitalar](https://github.com/Dioguint/sistema-hospitalar)
Sistema completo para gestão de estoque hospitalar, dividido em dois módulos:
- **Back-end CLI** em Python + MySQL com controle de inventário, lotes, validade e relatórios em CSV
- **Dashboard web** em HTML/CSS/JS com gráficos de movimentações e alertas de estoque crítico

> Desenvolvido com base em 6 anos de experiência prática no Hospital Geral de Itaguaí.

**Tecnologias:** `Python` `MySQL` `HTML5` `CSS3` `JavaScript` `Chart.js`

---

### ✅ [To-Do App](https://github.com/Dioguint/todo-react)
Aplicação de lista de tarefas construída em React para praticar os fundamentos da biblioteca:
- Adicionar, editar, concluir e remover tarefas
- Filtros por status (Todas / Pendentes / Concluídas)
- Persistência com localStorage

**Tecnologias:** `React 18` `useState` `useEffect` `CSS3`
